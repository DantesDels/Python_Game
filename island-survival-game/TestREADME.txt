# Island Survival Game
A console-based survival game where players manage vital resources while stranded on an island.

## Description
In this game, players must navigate the challenges of survival by managing their hunger, thirst, and energy levels. Each day, players can choose from various actions to sustain themselves and face random events that can impact their survival.

## Game Features
- Resource Management: Players must keep track of hunger, thirst, and energy levels.
- Daily Actions: Players can perform actions such as fishing, searching for water, sleeping, and exploring.
- Random Events: Encounter events like rain, animal encounters, and resource discoveries that can aid or hinder survival.
- Game Over Conditions: The game ends when critical resource levels are reached or after a set number of days.

## Installation
1. Clone the repository.
2. Install the required dependencies listed in `requirements.txt`.
3. Run the game using the command:
   ```
   python -m src
   ```

## Usage
- Start the game and follow the prompts to manage your resources and make strategic decisions.
- Save your progress at any time, and load it later to continue your adventure.
